# i18n Guide

## 구조
- 번역 메시지: `_locales/{locale}/messages.json`
- 언어 목록(설정 탭 드롭다운): `_locales/languages.json`
- 기본 언어: `ko`

## 새 언어 추가 방법
1. `_locales/ko/messages.json`을 복사해 `_locales/{locale}/messages.json`을 만듭니다.
2. 모든 키를 유지한 채 `message` 값만 번역합니다.
3. `_locales/languages.json`의 `locales` 배열에 항목을 추가합니다.

예시:
```json
{ "code": "ja", "label": "日本語" }
```

## 규칙
- 키 이름은 변경하지 않습니다.
- 메시지 키 누락 시 기본 언어(`ko`) 값으로 fallback 됩니다.
- 설정 탭의 언어 선택은 `auto`(브라우저 언어) 또는 명시 언어를 저장합니다.

## 관련 파일
- `background/background.js`: 언어 메타/메시지 로딩, locale 결정
- `shared/i18n.js`: popup/content 공용 `t()` 및 DOM 치환
- `popup/popup.html`, `popup/popup.js`: 설정 탭 언어 선택 UI
- `content/content.js`, `content/overlay.js`: 콘텐츠 오버레이 언어 갱신
